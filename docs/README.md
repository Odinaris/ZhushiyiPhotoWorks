## 文档命名规范

所有新增的MD文件都应该：
1. 使用英文命名，大写字母分隔单词
2. 包含描述性的文件名
3. 放置在此docs文件夹中

例如：
- `NEW_FEATURE_GUIDE.md` - 新功能指南
- `API_DOCUMENTATION.md` - API文档
- `USER_MANUAL.md` - 用户手册
