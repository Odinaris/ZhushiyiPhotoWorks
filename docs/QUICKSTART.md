# 快速启动指南 ⚡

> 5分钟快速体验摄影作品小程序

## 🎯 前置条件

- ✅ 已安装微信开发者工具
- ✅ 已注册小程序账号(有AppID)
- ✅ 项目已下载到本地

## 🚀 快速启动(5步)

### Step 1: 打开项目 (30秒)
```
1. 打开微信开发者工具
2. 选择"导入项目"
3. 目录选择: c:/Users/Administrator/WeChatProjects/photo-works
4. AppID填写您的小程序AppID
5. 点击"导入"
```

### Step 2: 开通云开发 (2分钟)
```
1. 点击工具栏"云开发"按钮
2. 点击"开通"
3. 环境名称: photo-works (或自定义)
4. 等待初始化完成
5. 复制环境ID(类似: photo-works-xxxxx)
```

### Step 3: 配置环境ID (10秒)
```
1. 打开 app.js
2. 找到第6行 cloudEnvId
3. 替换为您的环境ID
4. 保存文件
```

### Step 4: 创建数据库 (1分钟)
```
在云开发控制台 -> 数据库,依次创建:
1. users
2. photographer  
3. banners
4. categories
5. albums
6. contact

权限设置为: "仅创建者可读写"
```

### Step 5: 部署云函数 (1.5分钟)
```
右键以下云函数 -> 上传并部署:云端安装依赖

必需部署(前台):
✓ getUserRole
✓ getHomeData
✓ getCategories
✓ getAlbumsByCategory
✓ getAlbumDetail
✓ updateAlbumStats
✓ getContactInfo

必需部署(后台):
✓ getAdminStats
✓ adminBanners
✓ adminCategories
✓ adminAlbums
✓ adminProfile
✓ adminContact
```

## 🎉 完成!

点击"预览"或"真机调试"即可查看效果!

## 🔧 设置管理员(可选)

如需使用管理后台:

```
1. 手机扫码进入小程序
2. 打开云开发控制台 -> 数据库 -> users
3. 找到您的记录
4. 编辑 role 字段为: admin
5. 刷新小程序
6. 首页会显示"管理后台"入口
```

---

## 📝 初始化内容(可选)

进入管理后台后,按顺序设置:

### 1. 品牌信息
```
管理后台 -> 品牌信息
- 品牌名称: 您的工作室名称
- 个性签名: 一句话介绍
- 主要风格: 人像,风光,纪实
```

### 2. 创建分类
```
管理后台 -> 分类管理
- 婚纱摄影
- 人像写真
- 风光摄影
```

### 3. 上传轮播图
```
管理后台 -> 轮播图管理
- 添加3-5张精选图片
```

### 4. 添加作品集
```
管理后台 -> 作品集管理
- 创建第一个作品集
- 上传封面和作品图片
- 勾选"是否精选"
```

## ⚠️ 常见问题

### Q1: 云函数调用失败?
**A**: 检查云函数是否部署成功,查看云函数日志

### Q2: 数据库读取失败?
**A**: 检查集合是否创建,权限是否正确

### Q3: 看不到管理后台?
**A**: 确认 users 表中您的 role 是 admin

### Q4: 图片上传失败?
**A**: 检查图片大小,建议压缩后上传

### Q5: TabBar图标不显示?
**A**: 目前使用占位符,可替换 images 目录下的图标
